# Fraud Alert Event

## Overview

The fraud alert event is generated by the Fraud Detection system when claims score above fraud thresholds or when post-payment analysis identifies suspicious patterns. These events route high-risk cases to the Special Investigation Unit (SIU) for manual review.

## Details

Generated by Fraud Detection in two scenarios: pre-payment when claims score >0.82 (automatic hold for SIU review) or post-payment when historical pattern analysis detects aberrant billing, upcoding, unbundling, or phantom billing. The event includes the fraud score, specific patterns detected, affected claims, and recommended investigation actions. Claims scoring 0.65-0.82 proceed to adjudication but are tagged for post-payment review. Target false positive rate is below 8%.
