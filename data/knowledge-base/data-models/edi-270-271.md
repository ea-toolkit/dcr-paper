# EDI 270/271

## Overview

EDI 270/271 represents the inquiry/response transaction pair for real-time eligibility verification. Providers send 270 inquiries to verify member coverage, and the system responds with 271 transactions containing eligibility details.

## Details

Standard EDI transaction pair for real-time eligibility verification. EDI 270 is the eligibility inquiry request sent by providers to verify member coverage, benefits, and accumulators. EDI 271 is the response containing eligibility status, benefit details, accumulator balances, and coverage information. The Eligibility Service handles both sides of this transaction, processing 270 requests and generating 271 responses. Also supports batch processing for bulk eligibility verification.
