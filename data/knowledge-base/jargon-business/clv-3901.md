# CLV-3901

## Overview

CLV-3901 was an action item created after the March 2024 Claims Gateway outage to implement separate database connection pools for batch and real-time processing.

## Details

CLV-3901 was assigned to Priya Anand with a target completion in sprint 38. The action item addressed the root cause of the March 2024 outage by separating the database connection pools so that batch processing operations cannot exhaust all connections and prevent real-time operations from proceeding. This was identified as a known design issue that had been deprioritized in Q4 2023 but became critical after the incident. The work was completed and merged on April 2, 2024.
