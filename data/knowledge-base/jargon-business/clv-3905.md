# CLV-3905

## Overview

CLV-3905 improved error messages to include connection pool statistics when database connection issues occur, making diagnosis easier for on-call engineers.

## Details

CLV-3905 addressed the diagnostic challenge identified during the March 2024 incident where the on-call engineer initially misdiagnosed the problem because error messages were generic ('Connection is not available, request timed out after 30000ms'). The action item enhanced error logging to include connection pool utilization statistics and clearer indicators of connection pool exhaustion. This improvement helps on-call engineers quickly identify connection pool issues rather than assuming network problems. The work was completed and merged on March 20, 2024.
