# AS2 Protocol

## Overview

Secure protocol used for real-time EDI transmission with clearinghouses, complementing SFTP for batch processing. Provides secure, reliable document exchange for healthcare transactions.

## Details

AS2 (Applicability Statement 2) enables secure, real-time transmission of EDI documents with clearinghouses. Used alongside SFTP for batch processing, AS2 provides immediate transmission capabilities for time-sensitive healthcare transactions with built-in security and reliability features.
