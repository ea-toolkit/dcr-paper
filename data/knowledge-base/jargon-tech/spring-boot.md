# Spring Boot

## Overview

Enterprise Java framework providing production-ready features out of the box. Version 3.x is standardized across most Java services in the claims platform. Chosen for its comprehensive ecosystem and enterprise support.

## Details

Spring Boot 3.x serves as the primary application framework for Claims Gateway, Payment Engine, and Provider Directory services. Provides dependency injection, auto-configuration, actuator endpoints for monitoring, and extensive integration capabilities. The framework choice aligns with enterprise Java standards and provides robust production features including metrics, health checks, and configuration management.
