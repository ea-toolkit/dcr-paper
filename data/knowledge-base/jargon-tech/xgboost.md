# XGBoost

## Overview

Gradient boosting machine learning framework used by Fraud Detection team for building predictive models. Latest model deployment reduced false positive rates from 6.3% to 5.8%.

## Details

Advanced machine learning framework (Extreme Gradient Boosting) used by Clearview's Fraud Detection team to build and deploy fraud scoring models. The new XGBoost model was deployed in production during Q3 2025, achieving improved performance with false positive rates dropping from 6.3% to 5.8%. This represents a more sophisticated approach to fraud detection than previous models.
