# James Whitfield

## Overview

James Whitfield is the team lead for Provider Network, the team responsible for the Provider Directory system and provider credentialing processes. His team manages provider data quality and network relationships.

## Details

Named as the Provider Network team lead responsible for the Provider Directory system, which contains provider demographics, credentials, specialties, network status, and fee schedules. His team also manages the provider re-credentialing process required every 3 years per NCQA standards. The Provider Directory is critical for network status determination and fee schedule lookups during claims adjudication.
