# Clinical Reviewer Interface Project

## Overview

Project to develop a comprehensive review interface for clinical reviewers who evaluate pended claims and pre-authorization requests. The interface will replace current basic database query tools with proper workflow capabilities.

## Details

The project aims to build a proper clinical review interface to replace the current system described as 'database queries wrapped in a web form'. The new interface will display claim details, member history, clinical criteria, and provide an approve/deny workflow for clinical reviewers. The project is scheduled for Q1 2026 and will utilize similar backend APIs as the member portal but with interfaces designed for internal clinical staff rather than members. The project addresses the most common persona requesting tooling improvements alongside claims adjudicators.
