# Digital ID Card Redesign Process

## Overview

Process to transform the current static image ID card into a dynamic display showing member ID, group number, plan name, PCP (for HMO plans), and copay summary. Includes sharing functionality allowing members to save as image.

## Details

The redesign replaces the existing static image approach with a data-driven ID card that pulls current member information to display relevant details. For HMO members, the card shows their assigned Primary Care Physician. The card includes a copay summary for quick reference. Members can share the ID card by saving it as an image file. Apple Wallet integration was considered but descoped from the November launch and moved to the backlog as a separate project.
