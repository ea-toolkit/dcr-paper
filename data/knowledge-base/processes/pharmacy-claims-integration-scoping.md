# Pharmacy Claims Integration Scoping

## Overview

Early feasibility analysis for migrating pharmacy claims from legacy PharmaCore system to Clearview's main claims processing pipeline. Identified as Q4 2025 priority with formulary integration as key challenge.

## Details

Scoping project to assess the feasibility and requirements for migrating pharmacy claims processing from the legacy PharmaCore system to Clearview's main claims processing pipeline. Currently 50K pharmacy claims per quarter are processed separately on the legacy system. Key integration challenges include formulary management, which is not integrated with the main platform at all. The lack of formulary integration is expected to be a significant problem when attempting to bring pharmacy claims onto the main pipeline.
