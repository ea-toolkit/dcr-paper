# CMS DRG Grouper Definitions

## Overview

CMS publishes updated DRG grouper definitions annually each October for the new fiscal year. These definitions must be incorporated into any DRG grouper library to ensure accurate institutional claims reimbursement calculations.

## Details

The Centers for Medicare & Medicaid Services (CMS) releases updated DRG grouper definitions annually, typically in October for the following fiscal year. These updates include changes to DRG assignments, payment weights, and grouper logic based on evolving medical practices and cost data. Any DRG grouper library used by the rules engine must be updated with these new definitions to ensure accurate reimbursement calculations for institutional claims. Failure to update could result in incorrect payments to hospitals.
