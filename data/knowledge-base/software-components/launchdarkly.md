# LaunchDarkly

## Overview

Feature flag management platform currently used for configuration management but under review for cost optimization. The team is considering removing it due to limited usage and high cost.

## Details

LaunchDarkly provides feature flag capabilities for the claims platform, allowing runtime configuration changes and gradual feature rollouts. However, the document indicates the team is considering removing it because it's expensive and barely used, suggesting that the feature flagging needs may be simpler than what LaunchDarkly provides or that the cost-benefit analysis doesn't justify continued usage.
