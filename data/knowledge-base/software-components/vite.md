# Vite

## Overview

Modern frontend build tool used for the Member Portal React application. Provides fast development server and optimized production builds with hot module replacement during development.

## Details

Vite serves as the build tooling for the Member Portal frontend, handling React 18 with TypeScript compilation, development server, and production bundling. Chosen for its superior development experience with instant hot module replacement and fast builds compared to traditional webpack-based tooling. Integrates with the React 18 and TypeScript stack to provide optimized bundles for the member-facing application.
