# Provider Directory

## Overview

Provider Directory maintains master data for all contracted healthcare providers including demographics, credentials, network status, and fee schedules. It's kept current by the Provider Network team.

## Details

This system serves as the authoritative source for healthcare provider information including provider demographics, credentials, network participation status, and associated fee schedules. The Provider Network team is responsible for maintaining current and accurate provider data. There may also be credentialing workflow functionality tied to this system.
