# Provider Network

## Overview

Provider Network team owns the Provider Directory system and handles provider credentialing and contracting processes, maintaining accurate provider network information.

## Details

This team manages provider-related systems and processes including the Provider Directory master data system, provider credentialing workflows, and provider contracting activities. They ensure the provider network information remains current and accurate for claims processing and member services.
